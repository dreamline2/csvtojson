# Query Facebook Tool
Notice: Token will expire!! That's terrible!!

###Setup
1. Install [nodejs](http://nodejs.org/)
2. `cd` your directory
3. `npm install`
3. `npm start`



### Standard user usage flow
1. upload csv
2. csv convert json
3. query fb graph api
4. print result file

### FB API
version: `v2.2`
request url: `http://graph.facebook.com/v2.2/`
